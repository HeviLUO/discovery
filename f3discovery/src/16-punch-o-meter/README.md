# 冲压式流量计

在本节中，我们将使用板上的加速计。

这次我们要建造什么？一个冲压式流量计！我们将测量你刺拳的威力。实际上，你能达到的最大加速度，因为加速度是加速计测量的。
强度和加速度是成比例的，所以这是一个很好的近似值。

加速度计也内置在LSM303DLHC封装内。就像磁强计一样，它也可以通过I2C总线访问。
